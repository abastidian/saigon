Saigon: Backend Construction Toolkit
====================================

.. _introduction:


Welcome to the documentation for the Saigon package!. This project is currently
under active development and is not yet officially released. Its purpose is to provide a comprehensive
set of modular and reusable constructs designed to streamline the development of robust and scalable
backend systems, particularly within an AWS ecosystem.
